# CGPA Calculator

A responsive web application to calculate CGPA based on user-input grades and credits. Built using HTML, CSS, and JavaScript with real-time calculation and input validation.

🚀 Features
- Real-time CGPA calculation  
- Dynamic input handling  
- Input validation for accurate results  
- Clean and user-friendly interface  
- Responsive design for different screen sizes  

🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  

📌 How to Use
1. Enter subject grades  
2. Enter corresponding credits  
3. Click calculate  
4. View your CGPA instantly
